# Product Overview

Chatify is a full-stack real-time chat application with custom JWT authentication.

## Core Features

- Real-time messaging using Socket.io with online/offline presence indicators
- Custom JWT authentication (no third-party auth providers)
- Welcome emails sent on user signup via Resend
- Image uploads handled through Cloudinary
- Notification and typing sounds with user toggle controls
- API rate-limiting powered by Arcjet

## Target Deployment

Free-tier friendly deployment (designed for Sevalla), with MongoDB for data persistence.
